# Web ghost
Webghost is a web fingerprinting tool that pulls web technologies, CMS, WAF, and SSL certificates. 

## Usage
```bash
python3 webghost.py -S <target>
```

## Features
- 🖥️ Server detection
- 🐘 Software detection
- 📦 CMS detection
- 🛡️ Security headers analysis
- 🔥 WAF detection 
- 🤖 robots.txt reader
- 🔒 SSL certificate info

## Requirements
```bash
pip install requests
```

## Installation
```bash
git clone https://github.com/xuzarr/Webghost.git
cd Webghost
pip install requests
python3 webghost.py -S 
```
